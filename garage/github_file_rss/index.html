GitHub File Changes RSS Feed Bookmarklet

Every file on GitHub has an RSS feed for its changes. This bookmarklet helps you quickly get the RSS feed URL for any GitHub file you're viewing.

How to use:

  1. Drag the button below to your browser's bookmarks bar
  2. Navigate to any file on GitHub
  3. Click the bookmarklet in your bookmarks bar
  4. Copy the RSS feed URL from the popup

Drag this button to your bookmarks bar:

Get GitHub RSS Feed

Once installed, you can use this bookmarklet on any GitHub file page to quickly get its RSS feed URL.

Note:

The RSS feed URL will be in the format: https://github.com/[user]/[repo]/commits/[branch]/[path].atom

You can use this URL in any RSS reader to follow changes to the specific file.
